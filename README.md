# AI Tech Interviewer

AI Tech Interviewer is a Django-based web application that simulates real technical interviews.  
It allows users to select a technology, answer technical questions, and receive AI-powered evaluation with scoring.

---

## Features

- User Authentication (Login / Logout)
- Technology-based Interview Selection
- Dynamic Question Flow
- AI-based Answer Evaluation (Google Gemini)
- Automatic Score Calculation
- Result Summary Page
- Clean and Responsive UI

---

## Tech Stack

- Backend: Django 6
- AI Integration: Google Gemini (google-genai)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- Environment Variables: python-dotenv

---

## How It Works

1. User selects a technology (Frontend / Backend).
2. System generates technical questions.
3. User submits answers.
4. AI evaluates each response.
5. Final result displays total score and average score.
